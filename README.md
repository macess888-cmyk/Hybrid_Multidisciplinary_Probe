# Hybrid Multidisciplinary Probe

A multidisciplinary research and review workbench for discovering, preserving, classifying, pressure-testing, and connecting public technical, scientific, governmental, institutional, and architectural claims.

## Core Posture

- observation before explanation
- evidence before conclusion
- unknowns preserved
- classification proportional to evidence
- broad observation separated from technical verification
- convergence of independently supported observations preferred over consensus of opinion

## Primary Components

- methodology
- taxonomy
- ontology
- reusable inspection lenses
- source-specific workflows
- observation registries
- bounded campaigns
- evidence collections
- independent reviews
- maps, dashboards, reports, and receipts

## Core Invariant

No layer may silently inherit the authority, certainty, validity, or provenance of the layer before it.
